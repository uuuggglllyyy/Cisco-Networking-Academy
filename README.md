Этот репозиторий содержит исходный код веб-приложения для **онлайн-учебника по сетевым технологиям Cisco**. Проект создан для изучения основ построения и администрирования сетей на базе оборудования и стандартов Cisco.

Проект разработан на **Python** с использованием фреймворка **Django** и представляет собой полноценный веб-сайт с интерактивными главами, мультимедийным контентом и адаптивным дизайном.

## ✨ Особенности

*   **Полноценный веб-сайт:** Приложение построено на Django, что обеспечивает удобную навигацию и управление контентом.
*   **Структурированное обучение:** Материал разбит на главы (`chapters`), что позволяет изучать сети Cisco последовательно.
*   **Мультимедийный контент:** Папка `media` содержит изображения и другие ресурсы, которые делают учебный процесс наглядным.
*   **Современный интерфейс:** Использование HTML, CSS и JavaScript для создания адаптивного и интерактивного пользовательского опыта.
*   **Готовность к расширению:** Структура проекта позволяет легко добавлять новые главы, тесты и интерактивные элементы.
*   **Автоматизация:** Настроен GitHub Actions (`.github/workflows`) для автоматизации процессов сборки или тестирования.

## 🛠 Технологии

## 🚀 Установка и запуск

Следуйте этим шагам, чтобы запустить проект локально:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/uuuggglllyyy/Cisco-Networking-Academy.git
    cd Cisco-Networking-Academy
Создайте и активируйте виртуальное окружение (рекомендуется):

bash
python -m venv venv
source venv/bin/activate  # Для Linux/macOS
venv\Scripts\activate   # Для Windows
Установите зависимости:

bash
pip install -r requirements.txt
Выполните миграции базы данных:

bash
python manage.py migrate
(Опционально) Заполните базу данных начальными данными:

bash
python create_initial_data.py
Запустите сервер разработки:

bash
python manage.py runserver
Откройте браузер и перейдите по адресу http://127.0.0.1:8000/.
